Que
===

> Simple Background Job Processing in Elixir



## Installation

Add `que` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:que, "~> 0.1.0-alpha.1"}]
end
```

and add it to your list of `applications`:

```elixir
def application do
  [applications: [:que]]
end
```
